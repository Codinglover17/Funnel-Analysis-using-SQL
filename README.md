# Funnel Analysis using SQL

## 📊 Project Overview
This project analyzes user funnel stages using SQL.

## 🧠 Objective
To identify drop-offs and improve conversion rates.

## 🛠 Tools Used
- SQL Server (SSMS)

## 📂 Dataset
User activity across stages:
- Visit
- Signup
- Purchase

## 🔍 Analysis Performed
- Stage-wise user count
- Conversion rate calculation

## 📈 Insights
- Maximum drop at signup stage

## 🚀 How to Run
Run queries.sql in SQL Server
